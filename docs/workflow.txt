流程

0. 预定义

站方定义好一些上传账号，用于存放下载后的文件


1. 用户注册

注册 -> 激活 -> 使用

功能包括

注册 / 激活 / 登陆 / 登出 / 修改资料/密码 / 绑定 网盘 (初期只支持 百度网盘)

在初期需要邀请码


2. 任务
2.1用户 提交 任务的 hash ，
 系统检测hash的相关信息，检测后告知用户是否可以下载
 若已经下载过，则让用户选择网盘 和 填写存放路径

2.2 用户选择要存放的网盘和存放路径，开始任务

2.3 任务步骤
2.3.1 下载任务，
2.3.2 存放到站方的上传账号
2.3.3 复制到用户的上传账号
2.3.4 email 通知用户
